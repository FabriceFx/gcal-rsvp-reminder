# Changelog

Toutes les modifications importantes apportées à ce projet seront documentées dans ce fichier.

## [1.0.0] - 2026-07-19
### Ajouté (Added)
- Script initial pour l'envoi de rappels automatiques.
- Modèle HTML bilingue (FR/EN) pour les e-mails de relance.
- Fonction `ajoutTrigger` pour la mise en place du déclencheur quotidien.
- Anonymisation des logs pour protéger les données personnelles.
- Ajout d'une fonction `onOpen` pour exposer le menu "À propos".
- Intégration d'un design Premium pour l'e-mail avec Google Fonts (Outfit).
